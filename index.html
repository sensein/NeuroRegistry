How it works
  1. Clicking opens a pre-filled GitHub Issue with your YAML.
  2. CI validates, ingests, computes diffs against previous versions, runs alignment.
  3. New versions of changed classes/properties are minted; whole-schema snapshot archived.
  4. Registry version bumps and provenance is logged.
